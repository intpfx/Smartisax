# WebView Donor Inbox

Put local WebView donor packages here before running:

```bash
tools/r2-webview-donor-inbox-audit.py
```

Accepted file types:

- `.apk`
- `.apkm`
- `.apks`
- `.xapk`
- `.zip` archives that contain APK files

This inbox is for offline inspection only. Adding a file here does not make it
flash-ready and does not authorize a ROM build.
